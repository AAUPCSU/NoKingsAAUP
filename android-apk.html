Manually Installing NoKingsAAUP on Android

Note: We prefer if you requested to be added to the Early Access program and then install and update the app through the Google Play Store. But if you want to get started right away, you can download the APK file and install it manually.

Download the APK file to get started:

↓ Download APK

Installation Instructions

  • Step 1 - Enable installation from unknown sources in your Android settings
  • Step 2 - Navigate to your Downloads folder and locate the app-release.apk file
  • Step 3 - Tap on the APK file to begin installation
  • Step 4 - Review the permissions and tap "Install"
  • Step 5 - Wait for the installation to complete
  • Step 6 - Tap "Open" to launch the NoKingsAAUP app
  • Step 7 - Create an account or log in to start using the app
Note: When updating the app using the manual installation method, make sure to uninstall the previous version AND clear cache first (Settings > Apps > NoKingsAAUP > Storage > Clear Cache) to avoid any issues with the new version.
Note: If you encounter any issues during installation, please check that you have enabled installation from unknown sources. On most Android devices, this can be found in Settings > Security or Settings > Apps > Install unknown apps.

Troubleshooting

  • If the APK won't open, ensure you have "Unknown Sources" enabled in your security settings
  • Make sure you have enough storage space on your device
  • If installation fails, try downloading the APK again
  • For other issues, please contact us for support
